# 環境設定

## コマンドエイリアス

このユーザー環境では以下のコマンドエイリアスが設定されています：

| エイリアス | 実際のコマンド | 説明 |
|-----------|--------------|------|
| `cat` | `bat` | syntax highlighting付きcat代替 |
| `ls` | `eza` | modern ls代替 |
| `find` | `fd` | 高速find代替 |
| `cd` | `zoxide` | smart cd代替 |
| `grep` | `rg` | 高速検索 |

### 注意事項
- 必要に応じてフルパスでの実行も可能
