# AI生成の不要なコードを削除

1. mainブランチとのdiffをチェックし、このブランチで導入されたAI生成の不要なコード（slop）をすべて削除します。
2. mainブランチの乖離が大きい場合は、直前にAI生成で作成・編集したファイルの不要なコード（slop）をすべて削除します。

このプロセスを進めるためにサブエージェントを起動します。

削除対象:
- 人間が追加しないような過剰なコメント、またはファイルの他の部分と一貫性のないコメント
- そのコードベースの領域にとって異常な過剰な防御的チェックやtry/catchブロック（特に信頼済み/検証済みのコードパスから呼び出される場合）
- ファイルと一貫性のないその他のスタイル

最後に、変更内容を1〜3文でサマリーとして報告してください
